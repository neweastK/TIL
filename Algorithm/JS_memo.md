#### vs 파이썬

- `문자열 * 횟수` 는 파이썬만 가능

- JS는 `string.repeat(횟수)`로 반복

- 예시('abc' 세번 반복)

  - python

    ```python
    res = 'abc'*3  
    print(res) #'abcabcabc'
    ```

  - JavaScript

    ```js
    res = 'abc'.repeat(3)
    console.log(res) //'abcabcabc'
    ```
